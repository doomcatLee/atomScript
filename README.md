# Atom Package Automation Tool

*Installs the following atom packages:*
  1. File icons
  2. Minimap
  3. Minimap linter
  4. Platformio Terminal
  5. Emmet

*Also installs zsh prezto plugin*


### How to use
```
./startDay
```
